# Wrapper for rusprofile

Запрашивает информацию у rusprofile по inn и возвращет информацию о компании, сл вида:
```yaml
{
   {
   "inn":"4705051080",
   "kpp":"1104705002930",
   "fio":"Слесарев Дмитрий Андреевич",
   "company":"ООО \"Авторитет\""
   }
}
```
Доступный эндпоинт:
/inn/{inn}
Поднять в докере:
docker-compose up
